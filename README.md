# Rust works

## 特徴
- 式ベースの言語
- 静的型付け
- 安全性高い
- 高速

## Rustの実行方法
### コンパイル
```bash
rustc .\hello_world.rs
# .exeファイルができる
```

### 実行
```bash
.\hello_world.exe
```

## 用語
- `式（expression）`: 何かしらの値を返す
- `文（statement）`: 処理は実行するが値は返さない
